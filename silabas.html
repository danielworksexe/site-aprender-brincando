Como Contar as Sílabas?

Contar as sílabas de uma palavra é como descobrir quantos pedacinhos ela tem quando falamos!

Uma dica divertida é bater palmas para cada pedacinho que você fala. Vamos tentar?

Palavra: BOLA

Falamos: BO (👏 uma palma) - LA (👏 duas palmas)

Então, BOLA tem 2 sílabas!

Palavra: MACACO

Falamos: MA (👏) - CA (👏) - CO (👏)

MACACO tem 3 sílabas!

Outra forma é perceber quantas vezes você abre a boca para falar a palavra inteira. Cada vez que abre para um som principal é uma sílaba!

Palavra: SOL

Abrimos a boca uma vez só!

SOL tem 1 sílaba!

Lembre-se: toda sílaba precisa ter pelo menos uma vogal!
